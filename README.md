# Super Store Sales Analysis And Forecasting Dashboard

## Overview
This project is an interactive Power BI dashboard developed to analyze sales performance, profit trends, customer behavior, and regional business insights using Super Store sales data. The dashboard provides data-driven insights and forecasting visualizations to support business decision-making.

---

## Features
- KPI Analysis for Sales, Profit, and Orders
- Regional and State-wise Sales Analysis
- Category and Sub-category Performance Insights
- Sales Trend Visualization
- Profit Analysis Dashboard
- Interactive Filters and Slicers
- 15-Day Sales Forecasting
- Business Intelligence Visualizations

---

## Tools & Technologies Used
- Power BI
- Microsoft Excel
- Data Visualization
- Forecasting Techniques
- Business Intelligence Analytics

## Dashboard Insights
- Identified top-performing regions and categories
- Analyzed profit and sales trends over time
- Compared regional sales performance
